Name of your project:

KeenKeeper



A short description:

KeenKeeper is a personal relationship tracking web app where users can manage friends, view their details, and stay connected through calls, texts, and video interactions. Every interaction is recorded in a timeline and visualized through analytics to help maintain meaningful relationships.




Technologies you used:

HTML5, CSS (Tailwind CSS), JavaScript, React.js, Recharts




3 key features of your project:



Friend management system with detailed profile view and interaction options (call, text, video)

Real-time interaction timeline that tracks and displays communication history

Analytics dashboard with pie chart showing interaction distribution (calls, texts, videos)
